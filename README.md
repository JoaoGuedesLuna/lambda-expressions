<h1>Lambda λ</h1>
<p align="justify">
    O conceito de função lambda em Java está intimamente ligado ao de interfaces funcionais, que consistem em interfaces contendo apenas um método abstrato. Em Java, uma função lambda é expressa através da sintaxe <b>(argumentos) -> expressão</b>, onde os argumentos representam os parâmetros da função e a expressão constitui o corpo da função. Essa abordagem visa a declaração do método no mesmo contexto em que será empregado.
</p>
<p align="justify">
    Uma função lambda pode consistir em nenhum ou vários comandos. Se a função possuir apenas um comando, as chaves não são obrigatórias. Por outro lado, se a função contiver múltiplos comandos, as chaves são necessárias, e também é obrigatório incluir o comando "return" em casos de funções com retorno.
</p>

<h2>Interfaces Funcionais</h2>
<p align="justify">
    Interfaces funcionais em Java são interfaces que contêm apenas um único método abstrato. Elas são fundamentais para a implementação de programação funcional em Java, especialmente com o advento das expressões lambda introduzidas a partir do Java 8. Elas são usadas extensivamente em APIs como as APIs de Streams, que são uma parte essencial do Java para operações de processamento de coleções de dados de maneira funcional e concorrente.
</p>
